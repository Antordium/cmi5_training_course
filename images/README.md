## Image Guidelines

### Style Requirements
- **Pixel Art**: Use 8-bit/16-bit style for sprites and icons
- **Color Palette**: Match the game's FF-inspired palette:
  - Primary: `#00D4FF` (cyan)
  - Secondary: `#FFD700` (gold)
  - Accent: `#FF6B35` (orange)
  - Background: `#0a0a1a` (dark blue)
- **Transparency**: Use PNG with alpha for sprites

### Technical Requirements
- **Format**: PNG (preferred), WebP (alternative)
- **Resolution**: 2x for retina displays recommended
- **File Size**: Keep under 500KB per image
- **Naming**: Use lowercase with hyphens

### Screenshot Guidelines
- Capture at 1920x1080 or higher
- Highlight relevant UI elements with colored boxes
- Remove any sensitive/personal information
- Use consistent browser/application theme

## Placeholder Behavior

If images are missing, the course will:
1. Display colored placeholder boxes
2. Show the expected filename
3. Function normally without visual assets

## Creating Pixel Art

Recommended tools:
- **Aseprite** - Professional pixel art editor
- **Piskel** - Free online pixel art tool
- **GraphicsGale** - Free Windows pixel editor
- **Pixilart** - Free online with animation support
